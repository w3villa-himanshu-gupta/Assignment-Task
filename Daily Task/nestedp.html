1. HTML is a markup language for describing web documents (web pages).
    • HTML stands for Hyper Text Markup Language
    • A markup language is a set of markup tags
    • HTML documents are described by HTML tags

  2. Tags in Html
    1. Html Heading
      • h1 is the bigest heading and font size is greater than h2,h3,h4,h5,h6
      • h2 is second biggest heading after h1 and greater than h3,h4,h5,h6
      • h3 is the third number heading and greater than h4,h5,h6
      • h4 is medium type of heading and font is smaller than h3
      • h5 is the second smallest heading in heading tags
      • h6 is the smallest heading in heading tags

    2. Html Paragraphs
      1. With HTML, you cannot change the output by adding extra spaces
      2. The browser will remove extra spaces and extra lines when the page is displayed.
      3. Any number of spaces, and any number of new lines, count as only one space.

    3. Html Lists
      1. Ordered list
        1. type="1 ,The list items will be numbered with numbers (default)
        2. type="A",The list items will be numbered with uppercase letters
        3. type="a",The list items will be numbered with lowercase letters
        4. type='i',The list items will be numbered with roman numbers

      2. Unordered list
        • disc,Sets the list item marker to a bullet.
        • circle,Sets the list item marker to a circle.
        • square,Sets the list item marker to a square
        • none,The list items will not be marked

  3. Tables in html5 and html
    1. Tables are defined with the table tag.
    2. Tables are divided into table rows with the tr tag.
    3. A table row can also be divided into table headings with the th tag.
    4. Table data can contain all sorts of HTML elements like text, images, lists, other tables, etc.
